Can you write a Python function that calculates the Fibonacci sequence using memoization?

✦

Here's a clean implementation using a decorator for memoization:

from functools import lru_cache

@lru_cache(maxsize=None)
def fibonacci(n: int) -> int:
    """Return the nth Fibonacci number."""
    if n <= 1:
        return n
    return fibonacci(n - 1) + fibonacci(n - 2)

# Example usage
for i in range(10):
    print(f"F({i}) = {fibonacci(i)}")

Key points:

  • @lru_cache automatically memoizes results — no manual dictionary needed
  • Time complexity drops from O(2ⁿ) to O(n)
  • maxsize=None means unlimited cache (use a number to limit memory)

Here's the generator version — it yields Fibonacci numbers one at a time, which is memory-efficient for large sequences:

def fib_generator():
    """Yield Fibonacci numbers infinitely."""
    a, b = 0, 1
    while True:
        yield a
        a, b = b, a + b
